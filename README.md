# Simple Calculator -
It is a simple python based smart calculator 🤓
It performs 8 types of mathematical operations.

## Features✨
*Basic operations* - Addition, Subtraction, Multiplication, Division 
*Extra* - Table of any two entered numbers.
*Continuous run* - Program will run continuously till you exit the program 

## How to run/ Kaise chalaye:
1. Download the Repo
2. Run this command
   Calc.py

   ## Language - Python 3
